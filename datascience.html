What is Data Science ?

Data science is an interdisciplinary field that uses scientific methods, processes, algorithms and systems to extract knowledge and insights from data in various forms, both structured and unstructured, similar to data mining.
Data science is a "concept to unify statistics, data analysis, machine learning and their related methods" in order to "understand and analyze actual phenomena" with data. It employs techniques and theories drawn from many fields within the context of mathematics, statistics, information science, and computer science.

Data science is an interdisciplinary field that uses scientific methods, processes, algorithms and systems to extract knowledge and insights from data in various forms, both structured and unstructured, similar to data mining. Data science is a "concept to unify statistics, data analysis, machine learning and their related methods" in order to "understand and analyze actual phenomena" with data. It employs techniques and theories drawn from many fields within the context of mathematics, statistics, information science, and computer science.

Turing award winner Jim Gray imagined data science as a "fourth paradigm" of science (empirical, theoretical, computational and now data-driven) and asserted that "everything about science is changing because of the impact of information technology" and the data deluge.
When Harvard Business Review called it "The Sexiest Job of the 21st Century", the term "data science" became a buzzword, and is now often applied to business analytics, business intelligence, predictive modeling, or any arbitrary use of data, or used as a glamorized term for statistics. In many cases, earlier approaches and solutions are now simply rebranded as "data science" to be more attractive, which can cause the term to become "dilute[d] beyond usefulness." While many university programs now offer a data science degree, there exists no consensus on a definition or suitable curriculum contents. To its discredit, however, many data science and big data projects fail to deliver useful results, often as a result of poor management and utilization of resources.

Future of Data Science : 

Data science is all about using scientific methods, Machine learning algorithms, mathematical modeling to analyze massive amounts of data and extract meaningful insights, recognize market trends and streamline business practices.
Big Data used to be “the next big thing of the future” like a few years back. We are already living the future and Big Data is everywhere. Every industry has now realized the potential of Data Science and how it can help them. It is the need of the hour for “A Data Science approach” to run the business better or rather to survive.

Common tools used by Data Scientists - 

Data Analysis - 
1. The R Project for Statistical Computing is still the most popular. I agree with William that it's best used through R Studio.
2. Pandas is a set of Python libraries if you don't want to learn a new language
3. The Julia Language is an upcoming alternative to R. I spent a little bit of time learning it and would like to keep track of where the project goes.

Data Warehousing -
1. MySQL: It can comfortably handle datasets that are a few GBs. Don't prematurely go to Hive. MySQL is optimized to death and is super good at latency for running ad-hoc queries.
2. CSV Files: You'd be surprised how far you can get with using these as your primary storage.
3. Hive/Shark/Redshift: For when your big data is actually big. Hive can do giant joins while Redshift is better for latency but more limited in its joins.

Data Visualization - 
1. D3.js for pretty visualizations to put on the web.
2. Matplotlib for ad-hoc Python plotting.
3. ggplot2 for R..

Machine Learning - 
1. R for training models and using them directly or over Hive.
2. Scikit-learn.
3. Weka for trying out standard algorithms quickly on new datasets in the past but its hard to productionize anything with it.

Social Network Analysis - 
1. NetworkX is a pretty good Python library for SNA but isn't distributed.
2. Stanford Network Analysis Project.
3. Apache Giraph is an open-source implementation of the Google Pregel paper.
